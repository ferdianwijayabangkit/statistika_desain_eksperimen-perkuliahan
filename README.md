## ✨ Isi Repositori

- **Modul & Materi PDF**: Materi presentasi lengkap mengenai statistika inferensia dan desain eksperimen, disusun secara sistematis.
- **Silabus & RPS**: Rencana Pembelajaran Semester (RPS) sebagai panduan utama kompetensi perkuliahan.
- **Jadwal & Informasi**: Update terkait jadwal kelas, praktikum, ujian, dan pengumuman akademik lainnya.
- **Tautan Pendukung**: Kumpulan referensi eksternal untuk memperdalam pemahaman mengenai rancangan percobaan dan analisis data.

## 🔧 Format & Alat Bantu

- **Materi Perkuliahan**: `PDF (Portable Document Format)`
- **Monitoring Partisipasi**: `Microsoft Excel Online (via SharePoint)`
- **Platform Utama**: `GitHub`

## 🚀 Cara Mengakses Materi

### Prasyarat
- Koneksi internet yang stabil.
- Aplikasi pembaca PDF (seperti Adobe Acrobat Reader atau browser).
- Akun Microsoft institusi (untuk mengakses dokumen di SharePoint UNTIRTA).

### Langkah-langkah Penggunaan

1.  **Jelajahi Folder Repositori**
    Navigasi folder yang tersedia untuk menemukan materi berdasarkan topik mingguan atau modul eksperimen tertentu.

2.  **Unduh Materi**
    Klik pada file yang dibutuhkan, lalu gunakan fitur "Download" untuk menyimpan salinan di perangkat Anda.

3.  **Pantau Keaktifan**
    Gunakan link monitoring partisipasi secara berkala untuk memastikan data keaktifan Anda telah terekam dengan benar.
